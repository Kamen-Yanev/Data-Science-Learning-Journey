# Python Conditionals – Practice Exercises

This folder contains beginner Python exercises focused on
conditional logic (`if / elif / else`).

The exercises were inspired by university coursework and later
rewritten and extended for clarity and originality.

All exercises are contained in a single Jupyter notebook.

---

## Exercises Included

### Exercise 1 – Input Classification
Classifies user input as:
- empty input
- a number
- lowercase word
- uppercase word
- mixed-case word
- input containing non-letter characters

---

### Exercise 2 – Extended Input Classification
Adds support for:
- decimal numbers
- negative numbers
- improved edge-case handling

---

### Exercise 3 – Simple Password Validator
Validates a password based on:
- minimum length
- letters and numbers
- special characters
- no spaces

Provides clear feedback explaining why a password is invalid.
